# Goers-tech-intensive-program technical test
Goers node.js technical test built on:
1. Node
2. Express
3. Mongoose


### Local setup
1. Clone the repo

2. Install packages

```
$ npm install
```

3. Create a `.env` file, like this
```
PORT=3000
MONGO_URI=mongodb://localhost:3030/shopping-cart
```

4. To build and start
```
$ npm start
```
